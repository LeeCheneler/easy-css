Easy CSS

Easy CSS

A simple ITCSS framework designed to be the initial building blocks for a website. This was mainly written as a learning exercise, it is by no means original. But it is hopefully good enough for production sites to be built on top of it!

The source for this project can be found here.

Great inspiration was taken from the nebula-css framework written by a great CSS author Robert Smith. Check it out at http://rbrtsmith.com/nebula-css/

Grid

A grid consists of a block element, <div class="o-grid"> containing multiple inline elements with fixed percentage widths, <div class="o-grid__item u-1/2">. This eradicates the need for row elements that then contain column elements (like Bootstrap's highly successful grid). In turn this means that you don't need to use floats push and pull elements and rendering lists in a grid is much easier.

Example
<div class="o-grid">
  <div class="o-grid__item u-1/3">
    Grid item
  </div>
  <div class="o-grid__item u-1/3">
    Grid item
  </div>
  <div class="o-grid__item u-1/3">
    Grid item
  </div>
</div>

List

Lists are defined in the DOM in a similar pattern to the above grid. You create a list item; <ul class="o-list">. And you create list items like; <li class="o-list__item">. By default a list will have all of its styling stripped.

Example
<ul class="o-list">
  <li class="o-list__item">
    List item
  </li>
  <li class="o-list__item">
    List item
  </li>
  <li class="o-list__item">
    List item

Flag

Flags allow you to align content left and right, with a component taking up the room it needs, then the body stretching to fill the remaining space. By default the body and component are vertically center aligned. To define a flag you first create a block element and give it the class; .o-flag. Then you place two elements within the flag element and give them the .o-flag__component and .o-flag__body classes depending on what order you want them to appear.

Example
<div class="o-flag">
  <div class="o-flag__component">
    <img src="some-image.png" />
  </div>
  <div class="o-flag__body">
    Some body text
  </div>
</div>

There are also responsive flags that will stack at certain breakpoints. You build them exactly like you do any other flag, but you define at what screen size they start to be a flag, below that screen size they will display the body and component as blocks. The responsive flag classes that replace the .o-flag class are .o-flag@xs .o-flag@sm .o-flag@md .o-flag@lg.

Utilities

There are bunch of useful utilities included with Easy CSS.

Responsive Classes

Some classes have responsive variants. Easy CSS follows a common approach to responsive classes. Classes that have responsive variants are the .o-flag .u-{insert fraction}. The responsive breakpoints are the following: @xs @sm @md @lg @max-xs @max-sm @max-md @max-lg The breakpoints @xs @sm @md @lg apply the style above the breakpoint. The breakpoints @max-xs @max-sm @max-md @max-lg apply the style up to the breakpoint.
